**Expense Tracker Collab Project**  
Due date: February 18th, 11:59pm

Work in teams of 2-3 students to create a Python program that manages and tracks expenses. Each team will build an expense tracking system using **dictionaries** to store expense data such as **categories**, **amounts**, and **dates**. The goal is to use python dictionaries to organize and manage data and collaborate on Github.

**The expense tracker will allow users to:**

* Add new expense  
* View all expenses  
* Filter expenses by category  
* Calculate the total expenses  
* Delete specific expenses

**Team Roles:**

* One person handles adding expenses.  
* Another implements the calculation of total expenses.  
* Another works on filtering or deleting expenses.  
* **Work on GitHub by cloning the repository, creating branches, and submitting pull requests.**

**Extension (optional 4 marks):**

* **Create a feature to generate expense summaries by week, month, or year.**  
* Integrate data visualization using libraries like Matplotlib to display spending trends.
